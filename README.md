# DRM6 — 고등학생의 시간대별 하루 일상 경험 조사

> **Day Reconstruction Method (DRM)** 기반 시간대별 일상 경험 설문 시스템

[![GitHub Pages](https://img.shields.io/badge/Survey-Live-brightgreen?style=for-the-badge&logo=github)](https://edu-data.github.io/DRM6/)
[![Version](https://img.shields.io/badge/version-6.2-blue?style=for-the-badge)](https://github.com/edu-data/DRM6)

## 📋 개요

DRM6은 **일상재구성법(Day Reconstruction Method)**을 활용하여 고등학생의 하루를 시간대별로 재구성하고, 각 활동에서 느끼는 정서를 체계적으로 측정하는 연구용 설문 시스템입니다.

### DRM4 → DRM6 핵심 개선

| 항목 | DRM4 | DRM6 |
|------|------|------|
| **활동 분류** | 긍정/부정 이분법 (인위적) | 🌅☀️🌙 시간대별 자연스러운 회상 |
| **설문 흐름** | 긍정→부정→배경조사 | 활동기록→정서기록→인적사항 (2단계 분리) |
| **네비게이션** | 전체 활동 한번에 | **블록별 순차 이동** (오전→오후→저녁) |
| **정서 항목** | 6개 항목 | **9개 항목** (각성 수준 기반 설계) |
| **정서 평가** | 전체 활동 일괄 | **시간대별 블록 단위** 정서 기록 |
| **데이터 저장** | Positive/Negative 분리 | **Activities 통합** (TimeBlock 구분) |
| **반복측정** | 매회 전체 응답 | **인적사항 1회만** (localStorage 자동 스킵) |

## 🕐 설문 구조

### 3분할 시간대

| 시간대 | 범위 | 아이콘 |
|--------|------|--------|
| 오전·점심 | 오전 8시 ~ 오후 1시 | 🌅 |
| 오후 | 오후 1시 ~ 오후 6시 | ☀️ |
| 저녁 | 저녁 6시 ~ 11시 | 🌙 |

### 설문 흐름 (v6.2 블록별 순차 네비게이션)

```
1️⃣  안내문 + 핸드폰 번호 입력
     ↓
2️⃣  🌅 오전·점심 활동 기록 (1~5개)
     │  · 일화, 무슨 시간, 누구와, 어디서, 이유
     │  · 완료 후 → [다음: 오후 활동 기록 →]
     ↓
     ☀️ 오후 활동 기록 (1~5개)
     │  · 완료 후 → [다음: 저녁 활동 기록 →]
     ↓
     🌙 저녁 활동 기록 (1~5개)
     │  · 완료 후 → [다음: 정서 기록 →]
     ↓
3️⃣  시간대별 정서 평가 (9항목 × 7점 리커트)
     │  🌅 오전·점심 활동 정서 (1/3 시간대)
     │  → ☀️ 오후 활동 정서 (2/3 시간대)
     │  → 🌙 저녁 활동 정서 (3/3 시간대)
     │
     │  😊 긍정: 즐거운(고각성), 행복한(중각성), 편안한(저각성)
     │  😞 부정: 짜증나는(고각성), 부정적인(중각성), 무기력한(저각성)
     │  ✨ 의미·가치: 의미있는, 가치있는, 만족할만한
     ↓
4️⃣  인적사항 (성별, 소재지, 학교유형, 학년, 진로결정)
     ※ 반복측정 시 동일 핸드폰 번호 → 자동 스킵
```

### 블록별 순차 네비게이션 (v6.2)

- **탭 제한**: 아직 방문하지 않은 시간대 탭은 비활성화 (건너뛰기 방지)
- **블록별 이동 버튼**: 각 시간대에 전용 `← 이전` / `다음 →` 버튼 제공
- **정서 블록 뷰**: 정서 기록도 시간대별로 분리 (🌅→☀️→🌙 순차 이동)
- **블록 헤더 표시**: 현재 정서 기록 중인 시간대와 진행도 (1/3, 2/3, 3/3) 표시

### 활동 카드 구성

| 필드 | 설명 |
|------|------|
| 📝 일화 | "기억에 남는 활동은?" — 시간대별 맞춤 예시 |
| ⏰ 무슨 시간 | 교과시간(→ 과목 선택), 쉬는시간, 점심시간, 학교방과후시간, 동아리시간, 상담시간, 학원시간, 아침시간(등교 전), 저녁시간, 귀가후시간, 취침전, 기타 |
| 👥 누구와 | 혼자, 교사, 친구, 선후배, 가족, 성인(학원강사포함), 이웃/지인, 온라인 친구, AI, 기타 |
| 📍 어디서 | 교실, 도서실, 운동장/체육관, 학원, 집, 기숙사, 카페/식당, 공원/야외, 종교시설, 문화시설, 교통수단, 스마트기기(컴퓨터포함), 기타 |
| 💬 이유 | 시간대별 맞춤 예시 제공 |

### 교과시간 과목 옵션

> `교과시간` 선택 시 하위 과목 선택이 나타남 (전문교과 포함)

국어, 수학, 영어, 사회/역사, 과학, 음악, 미술, 체육, 정보, 한국사, 제2외국어, 진로선택과목, **전문교과**, 기타과목

### 학교유형 옵션

- 일반계고
- 특성화고 (경영고, 공고, 과학기술고, 농고 등)
- 기타

## 🔄 반복측정 지원

- **핸드폰 번호**를 응답자 식별 ID로 활용
- 인적사항은 **첫 응답 시 1회만** 입력 → `localStorage`에 저장
- 2차 이후 설문 시 인적사항 페이지 자동 스킵, 저장된 데이터 재사용
- Activities 시트에 **날짜(Date)**와 **요일(DayOfWeek)** 자동 기록 → 측정 시점 구분

## 📝 변경 이력

### v6.2 (2026-03-31)

- **블록별 순차 네비게이션**: 오전→오후→저녁 순서로 활동 기록 후 다음 블록으로 이동 (건너뛰기 방지)
- **정서 블록 뷰**: 정서 기록도 시간대별로 분리 (🌅→☀️→🌙), 블록 헤더에 진행도 표시
- **탭 제한**: 미방문 시간대 탭 비활성화 (opacity/pointer-events 제어)
- **전문교과 과목 이동**: `무슨 시간` 옵션에서 제거 → `교과시간` 하위 과목 선택으로 이동
- **제출 후 안내**: 성공적 제출 완료 후 "브라우저를 닫으셔도 됩니다" 메시지 표시
- **CSS 보완**: 탭 비활성 스타일, 정서 블록 인디케이터, 완료 메시지 애니메이션 추가

### v6.1 (2026-03-30)

- **시간대별 맞춤 예시**: 활동 일화 및 이유의 placeholder를 오전/오후/저녁에 맞게 차별화
- **학교유형 간소화**: 일반계고, 특성화고(경영고, 공고, 과학기술고, 농고 등), 기타 3가지로 정리
- **무슨시간 옵션 개선**: 전문교과·쉬는시간 추가, 주말/휴일 옵션 삭제, 취침전 순서 조정
- **일화 질문 개선**: "무엇을 했나요?" → "기억에 남는 활동은?"으로 변경
- **제출 안내 문구 추가**: 설문 완료 버튼 앞에 응답 제출 대기 안내 표시

### v6.0 (2026-03-28)

- **초기 구현**: DRM6 시간대별 일상 경험 조사 설문 시스템
- **반복측정**: 핸드폰 번호 기반 인적사항 1회 입력, localStorage 자동 스킵
- **시간대 기록**: 날짜(Date), 요일(DayOfWeek), 한국 시간대(KST) 고정
